# Getting Started with Markdn

Welcome to Markdn! This guide will help you understand the basics of using Markdn for your Blazor applications.

## What is Markdn?

Markdn is a modern, type-safe content management system that lets you write content in Markdown and automatically generates Blazor components from it.

## Key Features

- **Type-Safe Content Collections**: Define schemas for your content and get compile-time type safety
- **Markdown to Blazor**: Automatically convert Markdown files to Blazor Razor components
- **Frontmatter Support**: Use YAML frontmatter to add metadata to your content
- **Source Generation**: Leverage C# source generators for optimal performance

## Getting Started

1. Create a collections.json file
2. Add markdown files with frontmatter
3. Use the generated content service

That's it! You're now ready to use Markdn in your project.
