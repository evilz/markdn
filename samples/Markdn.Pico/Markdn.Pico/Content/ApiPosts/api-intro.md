# Introduction to Markdn Source Generators

This post demonstrates the new WebAPI features in Markdn source generators.

## Features

- Type-safe collections
- Auto-generated slugs
- WebAPI endpoints
- Multi-format support

The source generator makes it easy to work with content files in your Blazor applications!
